# Иконки расширения

## Генерация иконок

Для создания PNG иконок из SVG файла выполните:

```bash
./generate-icons.sh
```

Если автоматическая генерация не работает, используйте один из следующих способов:

### Вариант 1: Онлайн конвертер
1. Откройте https://cloudconvert.com/svg-to-png
2. Загрузите `icon.svg`
3. Создайте PNG для размеров: 16x16, 32x32, 48x48, 96x96, 128x128
4. Сохраните файлы как `icon-16.png`, `icon-32.png` и т.д.

### Вариант 2: В Xcode
1. Откройте проект в Xcode
2. File > New > File > Asset Catalog (если нет)
3. Добавьте Image Set
4. Перетащите иконки разных размеров

### Вариант 3: ImageMagick (если установлен)
```bash
brew install imagemagick
for size in 16 32 48 96 128; do
  convert -background none -resize ${size}x${size} icon.svg icon-${size}.png
done
```

## Требуемые размеры
- icon-16.png (16x16) - для меню
- icon-32.png (32x32) - для Retina меню
- icon-48.png (48x48) - для настроек
- icon-96.png (96x96) - для Retina настроек
- icon-128.png (128x128) - для App Store и высокого разрешения

## Дизайн иконки
Иконка представляет собой:
- Градиентный фон (фиолетовый #667eea → #764ba2)
- Символ сетевого соединения (узлы и линии)
- Текст "IPv6" внизу
- Иконка замка (безопасность) в правом верхнем углу
